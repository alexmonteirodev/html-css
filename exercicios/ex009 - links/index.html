usando link interno

para isso, você precisa de outra pagina usando a tag a e clica em ctrl + barra de espaço para escolher arquivo aqui, lembre-se de usar o rel next e prev para indicar pro google se é pra frente ou pra tras

lembrando que tem que informar o caminho caso a pagina esteja dentro de uma pasta, por exemplo:pagina 3
